# 🏗️ Arquitetura

## 🧩 Stack base
- **Front-end:** Next.js (React)
- **Back-end:** NestJS
- **Banco de dados:** PostgreSQL
- **Hospedagem:** Vercel
- **Autenticação:** NextAuth
- **ORM:** Prisma
- **Infra adicional:** Supabase

## 📊 Fluxo de dados
```
Usuário → Front → API → Banco → Retorno JSON
```
